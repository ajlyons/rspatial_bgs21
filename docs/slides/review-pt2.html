Tabular Data

data frames and matrices

several sample data frames that come with R (i.e., mtcars, quakes, iris)

import csv files with read.csv()

  • alternative: readr::read_csv()
  • For stubborn data frames, try the Import Dataset button in RStudio

peek inside a data frame with: head(), nrow(), names(), View(), etc.

grab a single column with $ (example: mtcars$wt)

subset row and columns with my_dataframe[rows-expression, columns-expression]

rows-expression, columns-expression can be
  • vector of integers (use for sorting also)
  • an expression that returns TRUE / FALSE values (one for each row)
  • column names

dplyr can do all of this and more (and a lot easier)!


Base R Plotting

Functions for quick visualization:

  • plot()
  • plot(…, add=TRUE) - overlays more data on the last plot
  • hist()
  • boxplot()

Top five optional arguments for plot() for geospatial data:

  • axes = TRUE
  • asp = 1
  • pch = 16
  • col = 'red'
  • main = 'My Plot Title'

For more plotting options look at packages like ggplot2, tmap, etc.,

Importing and Plotting Vector Data

sf data frames are essentially regular data frames with a ‘geometry’ column

import GIS data with: st_read(source, layer)

where source can be a:

  • a folder (e.g, containing Shapefiles, geodatabase)
  • file (e.g., geojson file, KML file)
  • database connection string (e.g., “PG:dbname=postgis”)

layer is a named layer

view layers in a source with st_layers()

code to import Shapefiles, KMLs, GeoJSON, Geodatabase, GeoPackage

plot() works with sf objects

  • to plot just the outline try: plot(ca_counties %>% st_geometry())
  • to alter symbology by an attribute field use plot(ca_counties["col_name"])

Common Manipulations for sf Objects

get the bounding box: st_bbox()

get the coordinates of a point layer as a 2-column matrix: st_coordinates()

view (or assign) the coordinate reference system: st_crs()

(re)project to a different crs: st_transform()

refer to CRS / projections using their EPSG number

convert a regular data frame to a sf object: st_as_sf()

export sf objects to standard GIS formats: st_write()

Spatial Queries

Tests for spatial proximity or relationships

  • st_intersects() Overlaps or touches
  • st_touches() Touches only
  • st_within() Within
  • st_contains() Contains
  • st_overlaps() Overlaps
  • st_is_within_distance() Within a distance

These functions return i) logical values, or ii) row numbers.

The results are packaged in a list object (when sparse = TRUE) or as a matrix (sparse = FALSE)


Using test results for queries

Row numbers in a list - use slice():

test_results_lst <- x_sf %>% st_touches(y_sf, sparse = TRUE)

y_features_touching_ith_feature_in_x <- y_sf %>% slice(test_results_lst[[i]])


Logical values in a matrix - use filter():

test_results_mat <- x_sf %>% st_touches(y_sf, sparse = FALSE)

y_features_touching_ith_feature_in_x <- y_sf %>% filter(test_results_mat[,i])


Spatial Queries using [square bracket notation]

xy_intersection <- x_sf[y_sf, ]

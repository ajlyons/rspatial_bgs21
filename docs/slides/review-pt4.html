Geoprocessing

Geoprocessing functions operate on the geometry of features

Functions fall into 3 categories:

  1. geometric measurements (area and length)
  2. creating new geometries
  3. tests for spatial proximity or relationships (covered under spatial queries)


Make Geometric Measurements

  • st_area() Area
  • st_length() Length / perimeter
  • st_distance() Distance between features

Results are returned in map units (but unit info is saved!)

Convert to other units using units::set_units()


Create new geometries

One sf input:

  • st_buffer()
  • st_centroid()
  • st_convex_hull()
  • st_voronoi()
  • st_simplify()

Two sf inputs:

  • st_union()
  • st_intersection()
  • st_difference()


Tests for spatial proximity or relationships

  • st_intersects() Overlaps or touches
  • st_touches() Touches only
  • st_within() Within
  • st_contains() Contains
  • st_overlaps() Overlaps
  • st_is_within_distance() Within a distance
